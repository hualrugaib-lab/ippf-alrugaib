⚠ الإعداد الأول · First-Time Setup
لم يتم ربط Firebase بعد. اختر:
  • وضع محلي: استخدام بدون مزامنة سحابية (البيانات على هذا الجهاز فقط)
  • إعداد Firebase: فتح صفحة الإعداد لربط حساب Firebase الخاص بك
احصل على الرمز من المسؤول (Admin)
أو · OR
ℹ️ للمسؤول: أول حساب يُنشأ يصبح Admin تلقائياً. ولإضافة أعضاء جدد، استخدم لوحة "إدارة المستخدمين" داخل النظام لتوليد رمز دعوة.
📋 خطوات إعداد Firebase (مرة واحدة):
1 أنشئ مشروع Firebase: اذهب إلى console.firebase.google.com وأنشئ مشروعاً جديداً (مجاني).
2 فعّل Authentication: Build → Authentication → Sign-in method → Email/Password → Enable.
3 فعّل Firestore: Build → Firestore Database → Create database → Start in test mode.
4 انسخ Config: Project Settings ⚙ → General → Your apps → Web app (</>) → انسخ object الـ firebaseConfig
